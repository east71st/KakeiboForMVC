<h2>KakeiboForMVCについて</h2>
<h3>【機能について】</h3>
<ol>
  <li>家計簿データの入力機能
    <ul>
      <li>入力履歴を確認しながら入力できます。</li>
      <li>入力したデータはデータべースに保存されます。</li>
    </ul>
  </li>
  <li>家計簿データの表示機能
    <ul>
      <li>データベースに保存された家計簿データを日付順にリスト表示します。</li>  
      <li>表示する家計簿データの条件をパネル(左縦帯)で操作して表示できます。</li>
      <li>表示されたデータの収支それぞれの合計額が表示されます。</li>
    </ul>
  </li>
  <li>家計簿データの修正機能
    <ul>
      <li>データベースに保存された家計簿データを修正できます。</li>
      <li>リスト表示されたデータを編集し更新ボタンを押すとその行のデータが更新されます。</li>
      <li>削除ボタンを押すとその行のデータがデータベースから削除されます。</li>
    </ul>
  </li>
  <li>家計簿データの集計機能
    <ul>
      <li>家計簿データを月別に集計することができます。</li>
      <li>月別の収入と費目別、収支の推移が表示されます。</li>
    </ul>
  </li>
  <li>費目データの登録と修正
    <ul>
      <li>データベースに登録済みの費目データを表示できます。</li>
      <li>費目名を新規に登録できます。</li>
      <li>登録済みの費目名を修正できます。</li>
    </ul>
  </li>
  <li>表示データのコピー機能
    <ul>
      <li>画面に表示されているリストデータをクリップボードにコピーできます。</li>
      <li>家計簿の入力、表示、修正と費目の編集画面に対応しています。</li>
    </ul>
  </li>
  <li>家計簿データのCSVファイルダウンロード機能
    <ul>
      <li>表示画面の家計簿データのCSVファイルをダウンロードできます。</li>
    </ul>
  </li>
</ol>
