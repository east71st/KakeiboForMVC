【機能について】
①家計簿データの入力機能
　・入力はマウスとタブキーを使わずに矢印キーとテンキーのみで操作ができます。（エンターキーでフォーカス移動）
　・入力履歴を確認しながら入力できます。
　・入力したデータはデータべース(MySQL)に保存されます。
②家計簿データの表示機能
　・データベースに保存された家計簿データを日付順にリスト表示します。
　・表示する家計簿データの条件をコンソール(左縦帯)で操作して表示できます。
　・表示されたデータの収支それぞれの合計額が表示されます。
③家計簿データの修正機能
　・データベースに保存された家計簿データを修正できます。
　・リスト表示されたデータを編集し修正ボタンを押すとその行のデータが更新されます。
　・削除ボタンを押すとその行のデータがデータベースから削除されます。
④費目データの登録と修正
　・データベースに登録済みの費目データを表示できます。
　・費目名を新規に登録できます。
　・登録済みの費目名を修正できます。
